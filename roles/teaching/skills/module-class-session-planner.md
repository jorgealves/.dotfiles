## Purpose
Produce ready-to-use session plans for live coding sessions, workshops, or lectures. Ensures sessions are paced, interactive, and tied to measurable learning objectives.

## When to Use
- User is preparing for a live class or workshop session
- User needs a structured agenda with timing
- User wants talking points, examples, and exercises for a specific topic

## What It Does
1. Structures the session into segments (intro, concepts, demo, practice, wrap-up)
2. Assigns time blocks to each segment
3. Prepares code examples and live-coding scenarios
4. Creates hands-on exercises with difficulty progression
5. Includes discussion questions and common student misconceptions to address

## Inputs
- Topic / technology area
- Session duration (e.g., 90 minutes, 3 hours)
- Student level and prior knowledge assumptions

## Output
- Minute-by-minute session plan
- Code examples (runnable, annotated)
- Student exercises with expected solutions
- Instructor notes and tips
